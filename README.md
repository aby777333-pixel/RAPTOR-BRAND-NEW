# GIORAPTOR

Official website for **GIORAPTOR** — forex broker technology provider offering complete brokerage infrastructure and multi-asset trading solutions.

## Stack

- Next.js 15 (App Router)
- React 18
- TypeScript
- Tailwind CSS 3

## Develop

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Build

```bash
npm run build
npm start
```

## Deploy

Deployed on Netlify with `@netlify/plugin-nextjs`.
